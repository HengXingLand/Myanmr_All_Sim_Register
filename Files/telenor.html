ဘာမွမလုပ္ေဆာင္မီ မွတ္ပုံတင္၏အေရွ႕အေနာက္ႏွစ္ဖက္ႏွင့္ မိမိမ်က္ႏွာအား ပုံေသခ်ာရိုက္ကူးထားလိုက္ပါ။

ပထမဦးစြာ "မွတ္ပုံတင္ရန္" ကိုႏွိပ္ပါ။
မိမိဖုန္းထဲမွာ Web Browser တစ္ခုထက္မက ရွိေနပါကေရြးခ်ယ္ခိုင္းပါလိမ့္မည္။
Chrome Browser ရွိပါက Chrome ကိုေရြးခ်ယ္ေပးလိုက္ပါ။
မရွိခဲ့လၽွင္အျခားအဆင္ေျပရာ Browser တစ္ခုခုကိုသာေရြးခ်ယ္ေပးလိုက္ပါ။
( အကယ္၍ မိမိတို႔ဖုန္းထဲမွာ Browser တစ္ခုသာရွိခဲ့လၽွင္ ေရြးခ်ယ္ရန္ေပၚလာမည္မဟုတ္ပါ။ )

Telenor ၏မွတ္ပုံတင္ရမယ့္ Website ကိုေရာက္သြားလၽွင္ "စတင္ရန္" ကိုႏွိပ္လိုက္ပါ။

ေနာက္တစ္မ်က္ႏွာတစ္ခုမွာ မိမိတို႔မွတ္ပုံတင္လိုသည့္ ဖုန္းနံပါတ္ကိုရိုက္ထည့္ၿပီး "Sign In" ကိုႏွိပ္ပါ။
ကိုယ္ထည့္လိုက္တဲ့ဖုန္းနံပါတ္ကို Verification ကုဒ္ပို႔လာပါလိမ့္မယ္။
ေပးပို႔လာေသာကုဒ္ကိုရိုက္ထည့္ၿပီး "Sign In" ကိုထပ္ႏွိပ္ေပးပါ။


"ေရွ႕သို႔" ကိုႏွိပ္ေပးလိုက္ပါ။

ရိုက္ကူးထားေသာမွတ္ပုံတင္၏အေရွ႕အေနာက္ႏွစ္ဖက္ႏွင့္မိမိမ်က္ႏွာအား "Choose File" တြင္အသီးသီးထည့္၍
အမွန္ျခစ္ရမည့္ကြက္လပ္၏ေဘးမ်ားတြင္ အမွန္ျခစ္တပ္ေပးၿပီး "အတည္ျပဳမည္" ကိုႏွိပ္ေပးပါ။


သင့္ Sim Card ကို မွတ္ပုံတင္ရန္ ေအာင္ျမင္စြာစာရင္းေပးသြင္းေပးၿပီးပါၿပီ။
မွတ္ပုံတင္းျခင္းေအာင္ျမင္၊ မေအာင္ျမင္သည္ကို Telenor မွစစ္ေဆးေပးၿပီးသည့္အခါ SMS ေပးပို႔လာပါလိမ့္မည္။


( Choose File ကိုႏွိပ္သည့္အခါ ပုံမ်ားေရြးရန္ေနရာသို႔ေရာက္မသြားပါက Chrome Browser ကို
အသုံးျပဳ၍ မွတ္ပုံတင္ရန္ အႀကံျပဳလိုပါသည္။
မိမိဖုန္းမွာ Chrome Browser မရွိေသးပါက App Store တစ္ခုခုမွ အခမဲ့ ေဒါင္းယူနိုင္ပါသည္။ )
