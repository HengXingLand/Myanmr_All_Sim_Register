ဘာမွမလုပ္ေဆာင္မီ မိမိမွတ္ပုံတင္၏ အေရွ႕အေနာက္ႏွစ္ဘက္အား ပုံေသခ်ာရိုက္ကူးထားလိုက္ပါ။
အြန္လိုင္းမွတဆင့္မွတ္ပုံတင္ျခင္းျဖစ္သည့္အတြက္ MPT မွ
400 MB Data Package အား လက္ေဆာင္အျဖစ္ အခမဲ့သုံးစြဲနိုင္ခြင့္ရမည္။

ပထမဦးစြာ "မွတ္ပုံတင္ရန္" ကိုႏွိပ္ပါ။
မိမိဖုန္းထဲမွာ Web Browser တစ္ခုထက္မက ရွိေနပါကေရြးခ်ယ္ခိုင္းပါလိမ့္မည္။
Chrome Browser ရွိပါက Chrome ကိုေရြးခ်ယ္ေပးလိုက္ပါ။
မရွိခဲ့လၽွင္အျခားအဆင္ေျပရာ Browser တစ္ခုခုကိုသာေရြးခ်ယ္ေပးလိုက္ပါ။
( အကယ္၍ မိမိတို႔ဖုန္းထဲမွာ Browser တစ္ခုသာရွိခဲ့လၽွင္ ေရြးခ်ယ္ရန္ေပၚလာမည္မဟုတ္ပါ။ )

MPT ၏မွတ္ပုံတင္ရမယ့္ Website ကိုေရာက္သြားလၽွင္ အေပၚဆုံး "မိုလ္ဘိုင္းဖုန္းနံပါတ္" ကြက္လပ္ထဲမွာ
မိမိတို႔မွတ္ပုံတင္လိုသည့္ ဖုန္းနံပါတ္ကိုရိုက္ထည္ပါ။
ၿပီးရင္ေအာက္တစ္ေၾကာင္းက အဝါေရာင္ေလးနဲ႔ "ကုဒ္ကိုရယူပါ" ဆိုတာေလးကိုႏွိပ္လိုက္ပါ။
ကိုယ္ေရးထည့္လိုက္တဲ့ဖုန္းနံပါတ္ကို Verification ကုဒ္ပို႔လာပါလိမ့္မယ္။
ေပးပို႔လာေသာကုဒ္ကို "အတည္ျပဳရန္ကုဒ္" ဆိုတဲ့ကြက္လပ္ထဲကိုရိုက္ထည့္ၿပီး
"အေကာင့္အသုံးျပဳရန္ဝင္ေရာက္သည္။" ကိုႏွိပ္ေပးပါ။

ဒါဆိုမွတ္ပုံတင္ရမယ့္စာမ်က္ႏွာကိုေရာက္သြားပါၿပီ။
"လူႀကီးမင္းမွတ္ပုံတင္၏အေရွ႕ဘက္ျခမ္းအားေပးပို႔ပါ" ကိုႏွိပ္ၿပီး
ရိုက္ထားတဲ့မိမိမွတ္ပုံတင္၏အေရွ႕ဘက္ျခမ္းကိုသြားေရြးထည့္ေပးပါ။
"လူႀကီးမင္းမွတ္ပုံတင္၏အေနာက္ဘက္ျခမ္းအားေပးပို႔ပါ" ကိုႏွိပ္ၿပီး
ရိုက္ထားတဲ့မိမိမွတ္ပုံတင္၏အေနာက္ဘက္ျခမ္းကိုသြားေရြးထည့္ေပးပါ။
အားလုံးေသခ်ာၿပီဆိုရင္ေတာ့ ေပးပို႔လို႔ရပါၿပီ။
ဒီေနရာမွာလိုင္းမေကာင္းရင္နည္းနည္းၾကာတတ္ပါတယ္၊ ေစာင့္ေပးလိုက္ပါ။

ၿပီးရင္ေအာက္ကိုနည္းနည္းဆင္းၿပီး "ျပင္ဆင္ေရးသားသည္" ဆိုတဲ့ခလုတ္ကိုရွာၿပီးႏွိပ္လိုက္ပါ။


"သုံးစြဲသူအမည္" ဆိုတဲ့ကြက္လပ္ထဲကို မိမိတို႔မွတ္ပုံတင္ထဲက နာမည္အတိုင္းေသခ်ာေရးထည့္ေပးပါ။
"နိုင္ငံသားစိစစ္ေရးကဒ္ျပားအမွတ္ / မွတ္ပုံတင္အမွတ္" ဆိုတဲ့ကြက္လပ္ထဲမွာေတာ့
မိမိတို႔ရဲ့မွတ္ပုံတင္ထဲကအတိုင္း မွတ္ပုံတင္အမွတ္ကို ေသခ်ာထည့္ေရးေပးလိုက္ပါ။
( ျမန္မာ - အဂၤလိပ္ မိမိအဆင္ေျပသလိုေရးထည့္နိုင္ပါတယ္။ )
အားလုံးေသခ်ာၿပီဆိုရင္ေတာ့ "အိုေက" ခလုတ္ကိုသာႏွိပ္ေပးလိုက္ပါ။


သင့္ Sim Card ကို မွတ္ပုံတင္ရန္ ေအာင္ျမင္စြာစာရင္းေပးသြင္းေပးၿပီးပါၿပီ။
မွတ္ပုံတင္းျခင္းေအာင္ျမင္၊ မေအာင္ျမင္သည္ကို MPT မွစစ္ေဆးေပးၿပီးသည့္အခါ SMS ေပးပို႔လာပါလိမ့္မည္။


( ပုံမ်ားထည့္ရန္ ႏွိပ္သည့္အခါ ပုံမ်ားေရြးရန္ေနရာသို႔ေရာက္မသြားပါက
Chrome Browser ကို အသုံးျပဳ၍ မွတ္ပုံတင္ရန္ အႀကံျပဳလိုပါသည္။
မိမိဖုန္းမွာ Chrome Browser မရွိေသးပါက App Store တစ္ခုခုမွ အခမဲ့ ေဒါင္းယူနိုင္ပါသည္။ )
